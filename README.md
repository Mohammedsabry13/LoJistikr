# D Lojistik Express — Global Logistics Website

A professional corporate website for **D Lojistik Express (Delma Logistics)**, an Istanbul-based global air freight and cargo shipping company founded in 2016.

## 🛠️ Tech Stack
- **HTML5 & CSS3** — semantic structure and custom styling
- **Bootstrap 5.3** — responsive grid, offcanvas mobile menu
- **WOW.js + Animate.css** — scroll-triggered animations
- **Bootstrap Icons & Font Awesome 6.7** — dual icon sets
- **Plus Jakarta Sans** — Google Fonts typography
- **Vanilla JavaScript** — navbar behavior, form validation

⚠️ This template was originally created by Mohammed Sabry
credit is required: github.com/Mohammedsabry13

## ✨ Features
- Fullscreen hero section with 4-slide background carousel

- Sticky navbar with offcanvas mobile menu
- About Us, Services & Industries sections
- Quote request form (Import / Export / Transit)
- Scroll-reveal animations throughout
- Footer with real contact info, social links & "Since 2016" watermark
- Fully responsive across all screen sizes
